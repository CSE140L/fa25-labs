## Simulators

We have some simulators provided for you to play around with to better understand some labs.

### Lab 2

[Robertson's Multiplier Simulator](https://cse140l.github.io/fa25-labs/lab2/robertsons_multiplier.html)

- Provides a tabular way to see what values should be in a register at a given timestep when multiplying two signed binary numbers

### Lab 4

[Vending Machine Simulator](https://cse140l.github.io/fa25-labs/lab4/vending_machine.html)

- Provides a vending machine UI to play with to behaviorally observe how the vending machine should function
